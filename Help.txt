Bazaar Lens 1.0.0 User Guide

   - Hold Ctrl key in the game to view monster or event information at the mouse pointer position;
   - Please ensure to run the program with administrator privileges;
   - If tesseract-ocr.exe is not installed in the default location, please re-select the installation location in the Set tesseract-ocr.exe Path option.   
   - If you encounter any bugs or other issues, please contact bazaarlens@gmail.com. Thank you!
