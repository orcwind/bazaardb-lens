# 截图区域配置说明

## 配置文件位置

截图区域配置在：`data/Json/position.json`

## 配置结构

```json
{
  "monster": {
    "icon": {
      "top_left": {"x": 1144, "y": 401},
      "top_right": {"x": 1403, "y": 399},
      "bottom_left": {"x": 1146, "y": 639},
      "bottom_right": {"x": 1411, "y": 641}
    },
    "name": {
      "top_left": {"x": 1457, "y": 365},
      "top_right": {"x": 1825, "y": 378},
      "bottom_left": {"x": 1467, "y": 474},
      "bottom_right": {"x": 1894, "y": 478}
    }
  },
  "item": {
    "icon": {...},
    "name": {...}
  }
}
```

## 如何设置截图区域

### 方法1：使用 position_finder.py 工具

运行项目根目录下的 `position_finder.py`：

```bash
python position_finder.py
```

按照提示：
- 按数字键 1-4 记录怪物图标的4个角
- 按数字键 5-8 记录怪物名称的4个角
- 按 q/w/e/r 记录物品图标的4个角
- 按 t/y/u/i 记录物品名称的4个角
- 按 's' 保存并退出

### 方法2：手动编辑 position.json

直接编辑 `data/Json/position.json` 文件，修改坐标值。

## 坐标说明

- **icon区域**：怪物/物品图标的4个角坐标（用于检测鼠标是否在图标上）
- **name区域**：怪物/物品名称的4个角坐标（用于OCR截图区域）

程序会根据鼠标位置和这些参考坐标，动态计算实际的截图区域。

## 注意事项

1. 坐标是屏幕绝对坐标（不是相对于窗口的）
2. 需要记录4个角（左上、右上、左下、右下）才能准确定位
3. 修改后需要重启程序才能生效
